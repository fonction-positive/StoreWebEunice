# 可选扩展功能 (Optional Extensions)

## 1. 概述
在核心功能（MVP）开发完成并稳定运行后，可根据业务需求逐步迭代以下扩展功能，以提升用户体验和平台运营能力。

## 2. 功能列表

### 2.1 商品评论系统
- **功能描述**：用户在确认收货后，可以对购买的商品进行评分（1-5星）和文字评价，支持上传买家秀图片。
- **实现要点**：
  - 新增 `Review` 表 (user, product, order_item, rating, content, images)。
  - 只有购买过该商品的用户才有权限评论。
  - 管理员可审核或删除违规评论。

### 2.2 收藏夹 / 愿望单
- **功能描述**：用户可将感兴趣但暂不购买的商品加入收藏夹。
- **实现要点**：
  - 新增 `Favorite` 表 (user, product)。
  - 商品详情页显示“心形”图标，点击切换收藏状态。
  - 个人中心增加“我的收藏”列表。

### 2.3 优惠券与折扣系统
- **功能描述**：
  - **满减活动**：满100减10。
  - **优惠券**：用户领取优惠券，结算时选择使用。
- **实现要点**：
  - 新增 `Coupon` 表 (code, discount_amount, min_spend, start_time, end_time)。
  - 新增 `UserCoupon` 表 (user, coupon, is_used)。
  - 结算 API 增加 `coupon_id` 参数，后端校验有效性并计算最终金额。

### 2.4 简单推荐系统
- **功能描述**：在首页或商品详情页底部推荐“猜你喜欢”或“相关商品”。
- **实现策略**：
  - **规则推荐 (初期)**：
    - 同分类推荐：推荐同一分类下的其他热销商品。
    - 新品推荐：推荐最新上架的商品。
  - **协同过滤 (进阶)**：基于用户历史浏览和购买记录，计算相似度进行推荐 (需引入数据分析模块)。

### 2.5 高级数据统计图表
- **功能描述**：为管理员提供更直观的数据可视化。
- **实现要点**：
  - 前端引入 `ECharts` 或 `Chart.js`。
  - 后端编写聚合查询 API (Aggregation)，返回 JSON 格式的统计数据。
  - 展示维度：
    - 过去 7 天销售额趋势。
    - 用户地域分布。
    - 商品销量 Top 10 排行榜。
    - 访问量 (PV/UV) 统计 (需记录访问日志)。

### 2.6 第三方登录
- **功能描述**：支持微信、QQ、Google 或 GitHub 快捷登录。
- **实现要点**：
  - 使用 OAuth 2.0 协议。
  - 后端集成 `django-allauth` 或手动实现 OAuth 回调处理。
  - 绑定第三方账号到现有 User 账户。

---

## 3. 扩展路线图 (Roadmap)
1. **v1.1**: 收藏夹 + 商品评论 (增强用户互动)。
2. **v1.2**: 优惠券系统 (促销工具)。
3. **v1.3**: 数据统计图表 (辅助运营决策)。
4. **v2.0**: 移动端 APP 适配 (React Native / Flutter) 或 小程序开发。
